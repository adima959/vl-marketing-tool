# CSS & Styling Patterns

Essential styling approach and design tokens.

**Strategy**: Ant theme + CSS Modules + Tailwind + CSS variables
**Tokens**: `styles/tokens.css` is single source of truth - never hardcode colors/spacing
**Ant Overrides**: CSS Modules with :global() + !important when theme config isn't enough
**Typography**: Use `tabular-nums` for table numbers, not monospace

See full token categories, CSS Module patterns, and component library guidance in original documentation or ask for specifics.
