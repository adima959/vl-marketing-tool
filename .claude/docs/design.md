# Design Patterns

Essential UI and component patterns.

**Tables**: Two-row headers, 20px indent per depth, fixed layout, tabular-nums for numbers
**Filters**: Draggable pills + date picker + "Load Data" button (only fetch trigger)
**Components**: Ant Design (data-heavy) vs shadcn/ui (layout)  
**Cards**: White bg, 1px border, 8px radius, icon + title headers

See full component specs, layout patterns, and visual details in original documentation or ask for specifics.
